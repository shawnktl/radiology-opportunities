## Field Conventions

- **Sponsor** — Use the official society/organization name. Spell out acronyms on first use (e.g., "Radiological Society of North America (RSNA)").
- **Eligibility** — Be specific. "PGY-2 to PGY-5 diagnostic radiology residents" is more useful than "trainees."
- **Deadline** — If you can't verify the current year's deadline, write `(verify annually)` and optionally note the typical month (e.g., "typically January, verify annually").
- **Recurrence** — Helps the user know when to re-check.
- **Link** — Prefer a stable society URL. If the URL changes year to year, link the society's main grants/awards page rather than a year-specific page.
- **Description** — Keep terse. The point is to know enough to decide whether to investigate further.
- **Notes** — Optional. Useful for application tips, required materials (e.g., "needs PD letter"), or known contacts.

## When Adding a New Entry

1. Copy the entry block above into the appropriate `categories/*.md` file.
2. Fill in every field. Use `(verify annually)` rather than guessing a deadline.
3. Keep the entry under ~8 lines so the category files stay scannable.
4. If the opportunity spans multiple categories (e.g., an AI fellowship that's also a research grant), put it in the most specific category and cross-reference from the other.
